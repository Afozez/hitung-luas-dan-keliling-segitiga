Hitung Luas dan Keliling Segitiga

Langkah-langkah :
  • 1. Pilih menu di sebelah kiri untuk menghitung luas atau keliling segitiga.
  • 2. Masukkan panjang alas dan tinggi untuk menghitung luas segitiga.
  • 3. Masukkan panjang ketiga sisi segitiga untuk menghitung keliling segitiga.
  • 4. Klik tombol hitung untuk mendapatkan hasilnya.

Hitung Luas Segitiga

    Rumus Luas Segitiga yaitu : L = 1/2 x a x t

    • Dimana
    • L = Luas
    • a = Panjang Alas
    • t = Tinggi

    Hitung Keliling Segitiga

      Rumus Keliling Segitiga yaitu : K = a + b + c

      Dimana

      • K = Keliling
      • a = Sisi A
      • b = Sisi B
      • c = Sisi C
